PIXI.sound.utils.resolveUrl (source)string static

Resolve a URL with different formats in glob pattern to a path based on the supported browser format. For instance: "sounds/music.{ogg,mp3}", would resolve to "sounds/music.ogg" if "ogg" support is found, otherwise, fallback to "sounds.music.mp3"

Name Type Description
source string | PIXI.loaders.Resource

Path to resolve or Resource, if a Resource object is provided, automatically updates the extension and url of that object.
Returns:
Type Description
string The format to resolve to
